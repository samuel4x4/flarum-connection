## Features
### SSO
* Login
* Creation of an account

### Users
* Update the password of a user
* Update the email of the password
* Associate a user to a group
* Delete of a user
* Get a user by id
* Search for a user

### Tags
* Create a tag
* Delete a tag
* Define specific rights for tag
* Get all the tags

### Groups
* Create a group
* Update a group
* Delete a group

### Discussions
* Create a discussion
* Update a discussion
* Delete a discussion

### Posts
* Create a post
* Update a post
* Delete a post
